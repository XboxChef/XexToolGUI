How to patch a xex file with a xexp patch file

 

 

If you have no xexp patch file you must download it from
xbuc.net or JQE360.com.

 

xbuc.net        JQE360.com

 

 

 

1.    Select the xex file for patching

2.    Select the xexp patching file

3.    Select the save location and file name

4.    Click the patch icon

 

The program works in the background when it is finished or 

there is an error, it will appear in the log window.

 

 

Important!
It may not be available space in the name
